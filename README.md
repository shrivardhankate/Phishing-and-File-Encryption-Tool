🔐 CyberShield – Web-Based Security Toolkit
CyberShield is a web-based cybersecurity mini-project developed using Python (Flask). The application integrates two core security modules: a heuristic-based phishing URL detection system and an AES-powered file encryption & decryption tool.
This project demonstrates practical implementation of web security principles, threat analysis techniques, and cryptographic data protection mechanisms within a structured backend architecture.

🛠 Tech Stack
Python 3.12
Flask (Backend Framework)
Cryptography (Fernet AES implementation)
HTML (Frontend Templates)
OS & File Handling Modules

🧠 Security Concepts Demonstrated
Heuristic-based phishing detection
Risk scoring methodology
Input validation & URL parsing
Symmetric encryption (AES)
Secure file handling      
Separation of upload and encrypted storage
Controlled file download via backend

⚠ Disclaimer
This tool is intended for educational and research purposes only.
The phishing detection module uses heuristic analysis and does not replace professional security systems.
